## Repository Structure

- **main branch**: Contains all student-facing materials (lessons and assignments)
- **Working branches**: Used by instructors for preparing and reviewing content
- **CODEOWNERS**: Ensures only instructors can approve changes to main
- **Branch Protection**: Automated reminders for content review

## Quick Start

### Students
```bash
git clone https://github.com/jce-kehila-2026/classroom.git
cd classroom
# All materials are on the main branch
```

### Instructors
```bash
git clone https://github.com/jce-kehila-2026/classroom.git
cd classroom
git checkout -b lesson-XX-topic-name
# Develop content, then create a PR to main
```
