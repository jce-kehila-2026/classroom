# Student Guide

## Welcome to the Classroom Repository

This repository contains lessons and assignments for your course. All materials are available on the `main` branch.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/jce-kehila-2026/classroom.git
cd classroom
```

### Stay Updated

As new lessons and assignments are released, you'll need to pull the latest changes:

```bash
git checkout main
git pull origin main
```

## Working on Assignments

### Creating Your Own Branch

When working on assignments, create your own branch to avoid conflicts:

```bash
git checkout main
git pull origin main
git checkout -b your-name/assignment-XX
```

### Submitting Your Work

Depending on the course requirements, you may:
- Submit via a separate assignment repository
- Create a pull request to your team's repository
- Submit files through the course management system

**Note**: This repository is primarily for reading course materials. Follow your instructor's specific submission guidelines for assignments.

## Repository Structure

- Lessons are organized by topic
- Each assignment has its own directory with instructions
- Check the README files in each directory for specific guidance

## Important Notes

- **Main branch only**: All course materials are on the `main` branch
- **Don't modify main**: If you need to experiment, create your own branch
- **Pull regularly**: New content is added throughout the course
- **Read instructions carefully**: Each assignment has specific requirements

## Getting Help

- Check assignment README files for detailed instructions
- Review lesson materials for concepts and examples
- Contact your instructor or teaching assistants with questions
- Collaborate with your team members on group projects

## Questions?

If you have questions about the course materials or repository access, please contact your instructor.
